## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS + Bootstrap
- 🔒 Google Authentication (for Admin)
- 🌐 React Router
- 📦 Axios
- ☁️ Backend: MongoDB Atlas (via Express.js)
- 🔗 Deployed on: Netlify (Frontend), Render (Backend)
